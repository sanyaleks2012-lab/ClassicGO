-- Overview

Проект ClassicGO воссоздает Scaleform HUD в CS:GO[cite: 2].

[IMPORTANT]: Будьте внимательны с путями установки.

-- File Configuration

Путь установки: migi/csgo[cite: 2].

Замените файлы при распаковке архива.

-- Build Instructions

1. Run MIGI as Administrator[cite: 2].

2. Press Build MIGI[cite: 2].

3. Steam Launch Options:

> -game migi/csgo -language english
