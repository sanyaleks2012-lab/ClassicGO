ClassicGO

Введение

ClassicGO — это глобальная модификация, использующая инструмент MIGI[cite: 2] для восстановления легендарного Scaleform HUD. Мы возвращаем интерфейс из классической эпохи CS:GO, сохраняя все современные функции игры.

Установка

Чтобы установить мод, перенесите файлы из архива в папку migi/csgo. При появлении окна подтверждения — выберите замену файлов.

Сборка MIGI

Запустите лаунчер MIGI и нажмите кнопку Build MIGI. Дождитесь окончания процесса, чтобы новые файлы HUD упаковались в игровые архивы.

Параметры запуска

Для корректной работы укажите следующие параметры в Steam:

-game migi/csgo -language english
