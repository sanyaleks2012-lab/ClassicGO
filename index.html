ClassicGO Restoration

Introduction

MIGI — это инструмент для объединения аддонов CS:GO[cite: 2]. Проект ClassicGO использует его для восстановления оригинального Scaleform HUD, заменяя стандартную Panorama UI.


СКАЧАТЬ ПАТЧ (OldHUD.7z)

Installation

Скопируйте содержимое архива в migi/csgo и подтвердите замену всех файлов.

MIGI Build

Запустите лаунчер и нажмите Build MIGI для упаковки файлов.

Launch options

Добавьте это в параметры запуска Steam:

-game migi/csgo -language english
