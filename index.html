Initialization

ClassicGO — это комплексная модификация, возвращающая классический Scaleform HUD в современную среду CS:GO.

Внимание! Не заменяйте оригинальные файлы игры напрямую. Используйте систему MIGI.

Directory Setup

Расположите файлы мода по следующему пути в папке с игрой:

.../csgo legacy/migi/csgo

При распаковке обязательно подтвердите перезапись существующих файлов.

Building Process

Запуск и компиляция через лаунчер MIGI:

Параметры запуска в Steam:

-game migi/csgo -language english
